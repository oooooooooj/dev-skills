## 🎯 快速开始

### 🌟 Node.js版本 (推荐，支持Windows)
```bash
# 进入技能库目录
cd ~/dev-skills

# 安装依赖
npm install

# 生成健身房项目
node scripts/generate-project.js --name "my-fitness-app" --business fitness

# 生成餐厅项目
node scripts/generate-project.js --name "restaurant-booking" --business restaurant

# 生成诊所项目
node scripts/generate-project.js --name "clinic-system" --business clinic
```

### 📦 Bash版本 (仅支持macOS/Linux)
```bash
# 进入技能库目录
cd ~/dev-skills

# 生成健身房项目
./scripts/generate-project.sh --name "my-fitness-app" --business fitness

# 生成餐厅项目
./scripts/generate-project.sh --name "restaurant-booking" --business restaurant

# 生成诊所项目
./scripts/generate-project.sh --name "clinic-system" --business clinic
```

### 2. 启动项目
```bash
# 进入项目目录
cd ~/my-fitness-app

# 运行初始化脚本 (Node.js版本自动生成对应平台的脚本)
./scripts/setup.sh  # macOS/Linux
# 或
scripts\setup.bat   # Windows

# 开始开发
npm run dev
```

## 🎨 自定义配置

### 业务配置
```typescript
// config/business.config.ts
export const businessConfig = {
  name: "我的健身房",
  type: "fitness",
  entity: "课程",
  capacityUnit: "人",
  timeSlotDuration: 90,
  maxCapacity: 20,
  
  booking: {
    maxAdvanceDays: 3,
    minAdvanceHours: 4,
    allowPartial: false,
  },
  
  ui: {
    primaryColor: "#ef4444",
    iconSet: "fitness",
    theme: "energy",
  }
};
```
